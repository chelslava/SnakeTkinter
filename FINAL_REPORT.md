## 🎯 Инструкции по использованию

### Запуск игры
```bash
# Windows
run_game.bat
# или
.\run_game.ps1

# Linux/Mac
source .venv/bin/activate
python main.py
```

### Обучение нейросети
1. Играйте в игре (ИИ собирает данные)
2. Откройте "📊 Аналитика"
3. Нажмите "Обучить нейросеть"

### Смена режима
1. Нажмите "🎮 Режимы"
2. Выберите нужный режим
3. Игра автоматически перезапустится
